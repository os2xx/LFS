1.3. Changelog

This is version arm64-r12.0-319-systemd of the Linux From Scratch book, dated February 27th, 2024. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via https://www.linuxfromscratch.org/mirrors.html.
